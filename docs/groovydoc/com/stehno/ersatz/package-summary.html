Package com.stehno.ersatz

Package com.stehno.ersatz Description

The Ersatz Mock Server API.
  • Interface Summary 
    Interface Description
    ClientRequest An abstraction around the underlying HTTP server request that aids in matching and working with requests.
    Expectations The Expectations interface is the root element of the expectation configuration, which provides the ability to define request expectations and responses for test interactions.
    Request Configuration interface for HTTP request expectations.
    RequestWithContent Expectation configuration for a request with body content.
    Response Used to configure the provided response to an HTTP request.
    ServerConfig Configuration interface for an Ersatz server instance.
  • Class Summary 
    Class Description
    AuthenticationConfig Configuration object for BASIC and DIGEST authentication support.
    ContentType A content-type representation.
    Cookie Ersatz abstraction of a request or response cookie.
    CookieMatcher Hamcrest matcher used to match Ersatz request cookie instances.
    Decoders Reusable request content decoder functions.
    DecodingContext Request-specific contextual information used by request content decoders.
    Encoders Reusable response content encoders.
    ErsatzMatchers Some reusable Hamcrest matchers useful in Ersatz expectations.
    ErsatzServer The main entry point for configuring an Ersatz server, which allows configuring of the expectations and management of the server itself.
    MultipartRequestContent Defines the request body content for a multipart request.
    MultipartRequestMatcher A Hamcrest matcher used to match MultipartRequestContent.
    MultipartResponseContent Response content object used to return a multipart response to a request.
    NoCookiesMatcher A Hamcrest matcher for matching the case when there should be no cookies configured in a request.
    RequestDecoders Configuration manager for a collection of request content decoders.
    ResponseEncoders Provides management of response encoders.
  • Enum Summary 
    Enum Description
    Authentication Enumerates the supported authentication types.
    HttpMethod Enumeration of the supported HTTP request methods.
