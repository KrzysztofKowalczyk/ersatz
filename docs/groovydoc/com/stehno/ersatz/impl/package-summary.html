Package com.stehno.ersatz.impl

Package com.stehno.ersatz.impl Description

The impl package provides the internal implementation classes for the API. Generally, test-writers will want to avoid directly using anything from this package.
  • Class Summary 
    Class Description
    DecoderChain A function chain for request decoders.
    EncoderChain A function chain for response encoders.
    ErsatzMultipartResponseContent Implementation of the multipart response content interface.
    ErsatzRequest Request implementation representing requests without body content.
    ErsatzRequestWithContent Ersatz implementation of a Request with request body content.
    ErsatzResponse Implementation of the Response interface.
    ExpectationsImpl Implementation of the Expectations interface.
    FunctionChain Abstraction for an object chain which will have method calls applied from "first" to "last" until a non-null result is returned.
    MultipartPart Representation of a multipart part for requests and responses.
    RequestMatcher Request-specific wrapper around hamcrest matchers to provide property-based matching based on request attributes.
    UndertowClientRequest The primary ClientRequest implementation used to wrap and expose the important parts of the underlying Undertow request context.
    UnmatchedRequestReport Helper object used to build and render a report of the unmatched request and the configured expectations.
