Package: com.stehno.ersatz.proxy

[Groovy] Class ErsatzProxy

    • com.stehno.ersatz.proxy.ErsatzProxy

    • Standalone simple proxy server useful for testing proxy configurations. At this time, only HTTP proxies are supported.

          Starts the proxy server on an ephemeral port on localhost, so that it does not collide with other running servers. Once the server is started (when this method returns) the actual port and url will be available via the server getter methods. This method does not need to be called if auto-start is not disabled.
