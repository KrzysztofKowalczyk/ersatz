Package: com.stehno.ersatz

[Groovy] Class MultipartRequestMatcher

      • com.stehno.ersatz.MultipartRequestMatcher

      • A Hamcrest matcher used to match MultipartRequestContent. The matcher may be created directly or by using the closure or consumer static configuration methods.
