Package: com.stehno.ersatz

[Groovy] Interface Expectations

  • The Expectations interface is the root element of the expectation configuration, which provides the ability to define request expectations and responses for test interactions. Internal expectation matching is done using Hamcrest Matchers - the methods without explicit Matches provide one as a convenience based on the property and value type (see method description). All configured matchers must match for a specific expectation to be considered a match and if there are multiple matching expectations, the first one configured will be the one considered as the match.
