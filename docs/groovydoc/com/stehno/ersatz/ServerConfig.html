      • ServerConfig autoStart(boolean enabled)

        Used to enable/disable the auto-start feature, which will start the server after any call to either of the expectations configuration methods. With this setting enabled, any other calls to the start() method are ignored. Further configuration is allowed. Auto-start is disabled by default.

        Returns:
        a reference to the server being configured
        Parameters:
        autoStart - whether or not auto-start is enabled
