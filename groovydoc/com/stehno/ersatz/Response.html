      • Response encoders(ResponseEncoders encoders)

        Registers the collection of shared encoders. Encoders registered on the response itself, will override these, but the shared encoders will override any matching global encoders.

      • java.lang.String getContent()

        Used to retrieve the configured response content. The content will be converted to a String based on the encoder configured for the content-type and content object type; if not encoder is found, the toString() method will be called on the content object. If no content exists, an empty string will be returned.

        Returns:
        the response content (encoded)
