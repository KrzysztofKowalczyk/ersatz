        • static boolean attrs(java.util.Map<java.lang.String, java.lang.Object> attrs, org.apache.commons.fileupload.FileItem fileItem)

          Used to determine whether or not the given FileItem has the specified attributes. The allowed attributes are: 'fieldName' to match the file field name, 'string' to match the item value as a string, 'fileName' to match the name of the file if the content is a file, 'contentType' to match the content type of the entry, 'bytes' to match the content as a byte array, and 'size' to match the item size value.

          Returns:
          true if the specified conditions are matched
          Parameters:
          attrs - the map of attributes to be matched
          fileItem - the multipart file item
