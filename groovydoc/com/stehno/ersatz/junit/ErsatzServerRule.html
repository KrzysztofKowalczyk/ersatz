[Groovy] Class ErsatzServerRule

      • com.stehno.ersatz.junit.ErsatzServerRule
      • All Implemented Interfaces and Traits:
        org.junit.rules.TestRule

        @groovy.transform.CompileStatic
class ErsatzServerRule
extends ErsatzServer

        JUnit Rule implementation to simplify the use of Ersatz in JUnit tests. It may be used with @Rule or @ClassRule. It's is a drop-in replacement for the ErsatzServer class which provides all of its configuration functionality and adds an automatic call to stop() after each test method. The start() method must still be called manually by the test methods.
