          • boolean matches(ClientRequest clientRequest)

            Used to determine whether or not the incoming client request matches this configured request. If there are configured conditions, they will override the default match conditions (except for path and request method matching, and only those configured conditions will be applied. The default conditions may be added back in using the Conditions functions. The default match criteria are:

            • The request methods must match.
            • The request paths must match.
            • The request query parameters must match (inclusive).
            • The incoming request headers must contain all of the configured headers (non-inclusive).
            • The incoming request cookies must contain all of the configured cookies (non-inclusive).
            • The incoming request body content must match the configured body content (after conversion).
            Returns:
            true if the incoming request matches the configured request
            Parameters:
            clientRequest - the incoming client request
